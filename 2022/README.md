# 2022
I completed every puzzle for 2022 in C#.
To run them:
1. Create a .NET console app in the 2022 directory and add `<StartupObject>AdventOfCode</StartupObject>` to the csproj file.
2. Create an Inputs directory (under 2022) and add input text files, called input01.txt ... input25.txt
3. Modify AdventOfCode.cs to run specific puzzles
